Dúvidas Frequentes

A Regra de 3 é usada para encontrar valores desconhecidos quando há uma relação de proporcionalidade entre 3 valores. Você precisa de 3 valores conhecidos para calcular o 4º.

O Modo Escuro é um modo de exibição que altera as cores do fundo e do texto para tons mais escuros, proporcionando uma experiência mais confortável para os olhos em ambientes com pouca luz.

O modo Daltonismo adapta as cores da página para torná-las mais acessíveis para pessoas com deficiência de percepção de cores, ajustando as paletas de cores para torná-las mais visíveis e compreensíveis.
